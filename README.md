# Модуль 3. Backend - Web сервер

## Цель урока:

1. Научимся создавать сервер

1. Модуль HTTP
2. Инструмент Nodemon
3. Обработка get, post, remove запросов
4. Фреймворк Express.js
5. Шаблонизатор EJS
